## Detailed Description

**Tab Favicon Labeler** helps developers and power users organize browser tabs by replacing favicons with custom colorful text labels.

### Key Features:
- **Custom Labels**: Create 6-character labels (displayed as 3+3 lines)
- **Color Coding**: Set custom background and text colors
- **URL Pattern Matching**: Use regex patterns to target specific sites
- **Environment Management**: Perfect for staging/production environments
- **One-Click Toggle**: Enable/disable extension instantly
- **Import/Export**: Backup and share configurations
- **Real-time Updates**: Changes apply immediately

### Perfect For:
- Web developers managing multiple environments
- QA testers working across different stages
- Anyone with many similar-looking tabs
- Teams needing consistent tab organization

### How It Works:
1. Define URL patterns (regex supported)
2. Set custom labels and colors
3. Extension automatically replaces matching tab favicons
4. Toggle on/off as needed

**Privacy-focused**: No data collection, works entirely offline.
